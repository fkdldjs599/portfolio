코멘트

◆개요

-변호사 상담,견적 서비스를 채팅을 통해 쉽게 받아볼 수 있는 서비스
-의뢰인일때 / 변호사일때의 메뉴와 마이페이지가 조금씩 다름
-포트폴리오에선 변호사인 경우 이용하는 메뉴를 표현
-의뢰인이 작성한 상담신청서를 변호사는 마이페이지 > 사건 수임하기에서 볼 수 있음
-상담신청서를 본 변호사는 견적서를 작성할 수 있고 견적서를 보낼때 마이페이지에서 충전한 캐시를 지불해야 보낼 수 있음

◆공통

-PC / 모바일 반응형으로 작업
-모바일 기준은 가로 1px ~ 960px까지 < 업체 요청

◆헤더

-로그인 방법은 카카오로그인으로
-서면종류는 항목별로 마우스를 올렸을때 어떤 종류가 있는지 한줄로 뜨게 요청
-서면작성 의뢰 메뉴는 홈화면일때 "의뢰할 서류 유형을 선택해주세요"로 이동 / 다른 페이지일땐 메인으로 이동
-받은 견적서 / 보낸 견적서 메뉴는 채팅으로 이동

◆푸터

-하단 푸터는 웹/모바일 고정
-모바일일땐 상단에 있던 메뉴가 하단에 나타남

◆메인

-메인 배너, 메인 리뷰는 swiper 라이브러리 사용
-서면종류는 항목별로 마우스를 올렸을때 어떤 종류가 있는지 한줄로 뜨게 요청
-웹 사이즈일땐 앱 다운 링크, 모바일일땐 웹 사이트 링크를 보내도록 함

◆마이페이지

-PC일땐 좌측에 서브메뉴, 오른쪽에 컨텐츠
-모바일일땐 서브메뉴가 제목 옆 햄버거바를 눌러야 나오는 메뉴로 변환


◆채팅

-채팅 내에서 결제요청, 거래확정 등 서비스 이용 가능
-모바일일땐 디바이스 전체에 채팅창이 꽉 찬 화면으로 요청
