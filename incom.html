코멘트

◆개요

-인천 지역에 대한 정보를 공유하는 커뮤니티 페이지
-웹에이전시에서 배운 기술들을 취합하여 만든 개인용 포트폴리오

◆공통

-PC / 모바일 반응형으로 작업
-헤더는 가로 1px ~ 768px에서 햄버거바로 변환
-로그인 상태일때 헤더에 나오는 내 프로필 사진과 닉네임을 누르면 마이페이지로 이동
-마이페이지에서 유저가 직접 올린 맛집,놀거리,부동산을 다른 회원들이 볼 수 있음

◆맛집, 부동산

-각 상세페이지에서 글을 올린 회원에게 채팅으로 문의할 수 있음
-맛집의 메뉴 리스트는 swiper 라이브러리 사용

◆놀거리

-놀거리를 누르면 팝업으로 정보를 볼 수 있음
-이미지는 swiper 라이브러리 사용
-팝업 내용이 길어지면 스크롤을 통해 볼 수 있음

◆마이페이지

-PC에서 나오는 왼쪽 메뉴는 모바일일때 서브제목을 누르면 나오게 함
-추가하기 폼에서 등록할 이미지를 선택하면 로고나 이미지를 미리볼 수 있음
-맛집 추가하기 폼에서 메뉴는 12개 이상 추가되지 않게 함

◆채팅

-모바일일때 채팅하기를 누르면 대화상대 리스트 페이지로 이동하며 PC일땐 대화상대와 대화창이 함께있는 페이지로 이동
-채팅 이미지를 클릭하면 자세히 볼 수 있는 이미지 팝업이 뜸
-채팅을 입력하거나 + 버튼으로 이미지를 삽입하면 채팅이 입력됨
