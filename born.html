◆개요

◆공통

-헤더의 큐알코드 팝업으로 헬스장 출입 가능
-채팅을 통해 소통할 수 있음


◆회원

-멤버쉽과 수강권을 구매할 수 있음
-수업 일정을 확인할 수 있음
-운동과 식단에 대해 정보를 얻고 기록할 수 있음
-트레이너 수업에 관해 리뷰를 남길 수 있음


◆트레이너

-수업 일정을 만들고 관리할 수 있음
-관리자에게 수익 정산 신청을 할 수 있음


◆관리자

-공지사항/이벤트를 등록할 수 있음
-회원들에게 일괄 문자나 팝업을 전송할 수 있음
-트레이너 등록과 관리가 가능
